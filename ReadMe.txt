How to run ?

First run the Server socket on Terminal.  command : python socket_server.py --> Server will connect to the localhost.

Next command : python socket_client.py --> GUI will ask for the USERNAME, After entering username client will be connected with the same name.

Entry : Destination Cient = Username for 1to1 communication
        Destination Cient = All for 1toN communication

Entry : Message : Message to be sent 

If the message is 1toN Client will receive the message 'All***' concatinated with the message. Client will get sender's username at the beginning of the text string.
In the case of 1to1 there will be only 'From' with the sender's username.
